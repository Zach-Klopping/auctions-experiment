The payoff table below shows what you will earn from the auction as a function of your value for the widget, your bid, and the bid of the other participant. Notice that you can change the value used to calculate the payoff table. The default is the value that has randomly been drawn for you. To change the value used to calculate the payoff table, use the pull-down menu. This does not change your value, but lets you see what payoff table looks like for the other participant if they draw a different value than you. Your value is highlighted in red on the pull-down menu so you can always easily find it.
